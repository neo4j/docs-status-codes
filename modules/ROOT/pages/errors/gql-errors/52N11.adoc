= 52N11

== Status description
error: procedure exception - generic topology procedure error. An unexpected error has occurred: `{ <<msg>> }`.

== Example scenario

For example, trying to set default number of primaries to 12:

[source,cypher]
----
CALL dbms.setDefaultAllocationNumbers(12,0);
----

The query returns a chain of errors where GQLSTATUS 52N11 is the cause of GQLSTATUS xref:errors/gql-errors/52N02.adoc[52N02]:

.GQLSTATUS error chain
[source, error]
----
52N11: procedure exception - generic topology procedure error. An unexpected error has occurred: Number of primaries '12' may not exceed 11 and needs to be at least 1
  52N02: procedure exception - procedure execution client error. Execution of the procedure dbms.setDefaultAllocationNumbers() failed due to a client error.
----
