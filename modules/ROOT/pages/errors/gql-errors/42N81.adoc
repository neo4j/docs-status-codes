= 42N81

== Status description
error: syntax error or access rule violation - missing request parameter. Expected `{ <<paramList>>1 }`, but got `{ <<paramList>>2 }`.

== Example scenario
Try running the following Cypher query without providing the parameters:

[source,cypher]
----
RETURN $a + $b + $c;
----

The query returns a chain of errors where GQLSTATUS 42N81 is the cause of GQLSTATUS xref:errors/gql-errors/42001.adoc[42001]:

.GQLSTATUS error chain
[source, error]
----
42N81: syntax error or access rule violation - missing request parameter. Expected $`a`, $`b` and $`c`, but got nothing.
  42001: syntax error or access rule violation - invalid syntax
----
