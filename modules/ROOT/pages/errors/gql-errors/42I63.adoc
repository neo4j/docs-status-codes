= 42I63

== Status description
error: syntax error or access rule violation - wrong subclause order. `ORDER BY`, `{ <<clause>> }` and `LIMIT` can only be used in this order in `RETURN`.

== Example scenario

Try using `ORDER BY` after `SKIP` and `LIMIT`:

[source,cypher]
----
UNWIND [3,5,1,3,7,10] AS x
RETURN x
SKIP 2
ORDER BY x;
----

The query returns a chain of errors where GQLSTATUS 42I63 is the cause of GQLSTATUS xref:errors/gql-errors/42001.adoc[42001]:

.GQLSTATUS error chain
[source, error]
----
42I63: syntax error or access rule violation - wrong subclause order. `ORDER BY`, `SKIP` and `LIMIT` can only be used in this order in `RETURN`. (line 2, column 1 (offset: 27))
"RETURN x"
 ^
  42001: syntax error or access rule violation - invalid syntax
----
