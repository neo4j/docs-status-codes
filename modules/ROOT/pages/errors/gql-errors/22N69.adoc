= 22N69

[role=label--changed-2025.07]
== Status description
error: data exception - index does not exist. The index `{ <<idxDescrOrName>> }` does not exist.


== Example scenario

Try dropping a nonexistent index:

[source,cypher]
----
DROP INDEX `nonExisting`;
----

The query returns a chain of errors where GQLSTATUS 22N69 is the cause of GQLSTATUS xref:errors/gql-errors/50N10.adoc[50N10]:

.GQLSTATUS error chain
[source, error]
----
22N69: data exception - index does not exist. The index 'nonExisting' does not exist.
  50N10: general processing exception - index drop failed. Unable to drop 'nonExisting'.
----
