= 42I68

== Status description
error: syntax error or access rule violation - mismatched pattern. Pattern, `{ <<input>>1 }`, does not match input, `{ <<input>>2 }`. Verify that the pattern is valid for constructing `{ <<valueType>> }`.

== Example scenario

Try parsing a duration string value:

[source,cypher]
----
RETURN duration("5 hours 12 minutes 15 seconds", "h 'hours' m 'minutes'") AS d;
----

The query returns a chain of errors where GQLSTATUS 42I68 is the cause of GQLSTATUS xref:errors/gql-errors/22007.adoc[22007]:

.GQLSTATUS error chain
[source, error]
----
42I68: syntax error or access rule violation - mismatched pattern. Pattern, `'h 'hours' m 'minutes''`, does not match input, `'5 hours 12 minutes 15 seconds'`. Verify that the pattern is valid for constructing `DURATION`.
  22007: data exception - invalid date, time, or datetime format
----
