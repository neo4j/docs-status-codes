= 22NAA

== Status description
error: data exception - invalid list for property-based access control rule. The expression `{ <<expr>> }` is not supported. Lists containing `{ <<exprType>> }` values are not supported for property-based access control.

== Example scenario

Try adding a property-based access control rule including a list with a `NULL` type:

[source,cypher]
----
GRANT TRAVERSE ON GRAPH * FOR (n) WHERE n.prop1 IN ["string value", NULL] TO role;
----

The query returns a chain of errors where GQLSTATUS 22NAA is the cause of GQLSTATUS xref:errors/gql-errors/22NA0.adoc[22NA0]:

.GQLSTATUS error chain
[source, error]
----
22NAA: data exception - invalid list for property-based access control rule. The expression 'n.prop1 IN ["string value", NULL]' is not supported. Lists containing NULL values are not supported for property-based access control.
  22NA0: data exception - invalid property-based access control rule. Failed to administer property rule.
----
