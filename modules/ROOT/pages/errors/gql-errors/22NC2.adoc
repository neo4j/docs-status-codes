= 22NC2

== Status description
error: data exception - node element type has no effect. The node element type `{ <<label>> }` must contain one or more implied labels, or at least one property type.

== Explanation
When defining a graph type, node element types must have some effect; otherwise, they are not allowed.

== Example scenario
Try setting a graph type as follows:

[source,cypher]
----
ALTER CURRENT GRAPH TYPE SET {
    (n:Node => )
};
----

The query returns a chain of errors where GQLSTATUS 22NC2 is the cause of GQLSTATUS xref:errors/gql-errors/42001.adoc[42001]:

.GQLSTATUS error chain
[source, error]
----
22NC2: data exception - node element type has no effect. The node element type `Node` must contain one or more implied labels, or at least one property type. (line 2, column 5 (offset: 35))
"    (n:Node => )"
     ^
  42001: syntax error or access rule violation - invalid syntax
----

To fix this, define at least one property type or add an implied label, for example:

[source,cypher]
----
ALTER CURRENT GRAPH TYPE SET {
    (n:Node => { prop :: ANY NOT NULL })
};
----
