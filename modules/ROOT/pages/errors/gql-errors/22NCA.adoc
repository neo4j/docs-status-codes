= 22NCA

== Status description
error: data exception - node element type already exists. A node element type identified by the label `{ <<label>> }` already exists in the graph type.

== Explanation

This error occurs when attempting to add a new node element type to the graph type, but a node element type with the same label already exists.

== Example scenario

Given the existing graph type:

[source,cypher]
----
ALTER CURRENT GRAPH TYPE SET {
    (p:Person => {name :: STRING })
};
----

Try adding a new node element type with the same label `:Person` as follows:

[source,cypher]
----
ALTER CURRENT GRAPH TYPE ADD {
    (p:Person => {name :: STRING, age :: INT })
};
----

The query returns a chain of errors where GQLSTATUS 22NCA is the cause of GQLSTATUS xref:errors/gql-errors/42001.adoc[42001]:

.GQLSTATUS error chain
[source, error]
----
22NCA: data exception - node element type already exists. A node element type identified by the label `Person` already exists in the graph type.
  42001: syntax error or access rule violation - invalid syntax
----

To fix this, change the query as follows:

[source, cypher]
----
ALTER CURRENT GRAPH TYPE ALTER {
    (p:Person => {name :: STRING, age :: INT })
};
----
