= 22N71

[role=label--changed-2025.07]
== Status description
error: data exception - duplicated index name. An index with the same name already exists: `{ <<idx>> }`.


== Example scenario

Try creating an index when there is another existing index with that name:

Given:
[source,cypher]
----
CREATE INDEX `myIndex` FOR (n:Person) ON (n.name);
----

Try creating another index with the same name:
[source,cypher]
----
CREATE INDEX `myIndex` FOR (n:Book) ON (n.title);
----

The query returns an error with GQLSTATUS 22N71 and status description:

.GQLSTATUS error
[source, error]
----
22N71: data exception - duplicated index name. An index with the same name already exists: `myIndex`
----
