= 53N33

== Status description
error: function exception - function invocation failed. Failed to invoke function `{ <<sig>> }` caused by: `{ <<msg>> }`

== Example scenario

Try parsing a duration string value:

[source,cypher]
----
RETURN duration({hours: 1}, "hh-mm-ss") as r;
----

The query returns an errors with GQLSTATUS 53N33 and status description:

.GQLSTATUS error
[source, error]
53N33: function exception - function invocation failed. Failed to invoke function duration(input :: ANY, pattern = DEFAULT_TEMPORAL_ARGUMENT :: STRING) :: DURATION caused by: A pattern can only be used in conjunction with a `STRING` input.
----
