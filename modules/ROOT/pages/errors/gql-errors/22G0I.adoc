= 22G0I

== Status description
error: data exception - invalid duration field. `{ <<field>> }` is not a valid duration field.

== Example scenario

Try running the following invalid query:

[source,cypher]
----
RETURN duration({dogYears: datetime()});
----

The query returns an error with GQLSTATUS 22G0I and status description:

.GQLSTATUS error
[source, error]
----
22G0I: data exception - invalid duration field. `dogYears` is not a valid duration field.
----
